Hawk IDS
Type Mobile Application for Intrusion Detection System

Back-end Development C++, Snort, Nmap, Barnyard2, XAMPP, MySQL

Repository GitHub (private)

Hawk IDS is a lightweight, minimalist, intrusion detection system specifically made for mobile phones. It is a straightforward and easy to learn application that provides the basic functionalities for tracking and analyzing network packets.

Main Features
  1. Live Network Traffic
    A real-time view and reportings of the recent network traffic coming into your device.
  2. Historical Network Data
    A view and reportings of network traffic that has entered your device up to 3 months.
  3. Quarantine Control
    A network controller to quarantine specified network traffics as defined by the user.
  4. Notifications
    A built-in notifier to alert user on possible network intrusion as defined by the user.
